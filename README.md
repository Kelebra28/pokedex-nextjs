# Pokedex con Nextjs, Redux y Jest

Es una pokedex, donde puedes ver ciertos detalles de los pokemones existentes en la **"Pokeapi"** en orden.

## Instalar 

```
yarn add
```

## Levantar App

```
yarn dev
```

## Correst Test

```
yarn run test
```

### Espesificaciones
- Nextjs
- Redux
- Jest Testing
- StyleComponets
- Vanilla Css
- React Hooks
